# BBS Services Web

Sitio web creado con Next.js, Tailwind CSS y TypeScript.

## Desarrollo

```bash
npm.cmd install
npm.cmd run dev
```

Luego abre `http://localhost:3000`.

## Produccion

```bash
npm.cmd run build
npm.cmd start
```

## Proximos pasos

- Conectar repositorio remoto en GitHub.
- Publicar hosting en Vercel u otro proveedor.
